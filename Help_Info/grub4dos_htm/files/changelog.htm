Grub4dos Guide - Changelog

Version 1.4 (17th May 2009)

  • Added Gfxmenu section following some subtle prompting from jaclaz.
  • Added Acknowledgements section.
  • Included README_GRUB4DOS.txt (as requested by Icecube - see here).
  • Edited Device Allocation/Numbering (as suggested by Icecube - see here).

Version 1.3 (22nd April 2009)

  • Updated guide for use with Grub4dos version 0.4.4 Final (build 2009-03-31).
  • Significant changes made to Installing Grub4dos section - which has been renamed as Loading/Installing Grub4dos.
  • Added Embedded Menu section.
  • Added Bootlace Commands section.
  • Added Grubinst Commands section.
  • Edited Introduction - simplifying the grub4dos boot process.
  • Updated Display Options in Configuration File Entries section.
  • Added Using && and ||Commands to Configuration File Entries section.
  • Amended/updated color, dd, geometry and map commands in Commands section.
  • Minor edits and spelling corrections throughout the guide.

Thanks to everyone who posted feedback on the Grub4dos Guide thread at boot-land.
  • Updated guide for use with Grub4dos version 0.4.4 (build 2009-02-05).
  • Added Load Via (Legacy) Grub and Load From Linux (kexec) to Installing Grub4dos section.
  • Added Renaming grldr section.
  • Added PXE Booting section.
  • Added Adding (Help) Messages to Menu Entries to Configuration File Entries section.
  • Added checkrange, checktime, dd, ls, tpm and uuid to Commands section.
  • Amended find, geometry, initrd, makeactive, map, setvbe and write commands in Commands section.
  • Removed testload from Commands section.
  • Minor edits and spelling corrections throughout the guide.
